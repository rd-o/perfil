\section{Antecedentes}
La visión artificial es un subcampo de la inteligencia artificial. El propósito de la visión artificial es que un sistema programado con esta tecnología pueda \guillemotleft entender\guillemotright  las imágenes que capta de su entorno. Un robot autónomo tiene un comportamiento y realiza tareas con alto grado de autonomía, esto es particularmente necesario en exploración espacial, fabricación, limpieza, etc.

Un robot o entidad virtual, para poder desenvolverse en su entorno, necesita información de entrada, que debe ser procesada para luego activar sus actuadores de acuerdo a los datos de entrada procesados. Estos datos de entrada pueden ser datos de distintos tipos de sensores. Dependiendo la complejidad del robot, estos pueden ser de sonido, imágenes y tacto. En cuanto a los sensores de imágenes y vídeo, para poder ser procesados es necesario tener un sistema que maneje estas imágenes y luego entienda el contenido de cada una de las imágenes que se le presente.

El proyecto estará orientado a desarrollar un sistema de navegación autónomo usando las bibliotecas de OpenCV (Open Source Computer Vision) para un robot. El robot a construir es un tipo, con dos orugas. Este tipo de configuración permite al robot una movilidad en todas las direcciones, y también en terrenos de tierra, grava y arena.

El robot necesita una plataforma independiente para poder desenvolverse, por lo tanto se usara un ordenador de placa única de bajo costo y bajo consumo de energía. 

El proyecto estará orientado al área de inteligencia artificial y robótica.

\textbf{Problema:} La necesidad de un sistema que permita a un robot poder desplazarse de forma autónoma, en un entorno alejado o de difícil acceso.

\subsection{Definición del problema}

Un \emph{entorno alejado} puede estar a cientos o miles de kilómetros alejado de los operadores. En entornos extremadamente alejados normalmente es difícil poder controlar un robot en tiempo real, ya que las señales enviadas pueden tardar en llegar a su destino.

Un \emph{entorno de difícil acceso} puede ser un pasadizo o cueva de difícil acceso para un operador. En este tipo de entorno puede que no haya visión directa desde el operador hacia el robot, por este motivo es casi imposible poder operar el robot. En ciertos casos en que se tenga un robot con una cámara inalámbrica, puede que la señal de control sea muy baja, debido a obstáculos y distancia alejada, esto puede ocasionar la perdida de la señal de vídeo y control.

\section{Objetivo del proyecto}
\subsection{Objetivos de la aplicación}
\begin{itemize}
  \item Se diseñara un robot que sea capaz de soportar la plataforma de hardware necesaria para el sistema.
  \item Se diseñara un robot de tal manera que sea totalmente inalámbrico.
  \item Se implementara un sistema de visión artificial que sea capaz de reconocer obstáculos al frente del robot.
  \item Se implementara un sistema de navegación que permita al robot moverse de forma autónoma a un destino establecido.
  \item Se implementara un sistema de visión artificial que sea capaz de distinguir obstáculos existentes en un plano.
  \item Se implementara un sistema de protección que alarme al robot en caso de estar peligrosamente inclinado
  \item Se implementara un sistema de comunicaciones, para que el robot pueda recibir las coordenadas de destino y otros datos. 
  \item Se implementara un algoritmo para detectar la profundidad de los obstáculos.
  \item Se desarrollara un sistema de posicionamiento, para detectar la posición del robot.

\end{itemize}

\subsection{Objetivos metodológicos}
Los objetivos a cumplir durante el desarrollo del proyecto son:
\begin{itemize}
  \item Se estimaran los costos y tiempo de desarrollo del proyecto.
  \item Se realizara investigación acerca de las tecnologías relacionadas al proyecto.
  \item Se realizara una estimación de cada historia de usuario del product backlog.
  \item Se asignara una prioridad a cada historia de usuario, para luego ejecutar primero las historias con mas prioridad.
  \item 
\end{itemize}

\section{Alcance}
El proyecto tendrá el siguiente alcance:
\begin{itemize}
  \item El robot podrá desplazarse en un terreno con obstáculos, y sera capaz de esquivar obstáculos o pasarlos por encima.
  \item El robot podrá desplazarse por terrenos planos, y también por pendientes no mayores a 25 grados. 
  \item El software de navegación recibirá coordenadas para llegar a un destino establecido por un operador, y este software sera capaz de conducir al robot a su destino de forma autónoma.
  \item El robot contara con una cámara de vídeo, la imagen obtenida por esta sera procesada por el sistema de visión artificial.
  \item Por limitaciones del hardware a utilizar, las obstáculos del terreno tienen que tener colores diferentes al terreno, para facilitar el trabajo al sistema de visión artificial.
  \item La cobertura entre el operador y el robot estará limitada por el máximo alcance del sistema de comunicación inalámbrica a utilizar.
  \item El sistema de visión artificial operara solo en ambientes con iluminación suficiente como para poder distinguir los obstáculos.
\end{itemize}

\section{Justificación}
Existen diferentes aplicaciones en las que este proyecto puede ser útil, por ejemplo en exploración espacial. En estos casos el robot esta alejado de su operador por distancias astronómicas, por tanto, el control del robot no es en tiempo real, y además hay posibilidad de que las comunicaciones se corten de vez en cuando, dejando al robot estático o incluso en riesgo de dejarlo inutilizable.

Otra aplicación para este proyecto es la exploración de lugares de difícil acceso para una persona, estos lugares pueden ser cuevas o lugares con contaminación de materiales radioactivos.

En nuestro país difícilmente se pueden encontrar proyectos relacionados a la inteligencia artificial y robótica. Este proyecto de grado pretende aportar a la comunidad universitaria con investigación relacionada a las áreas de este proyecto.
