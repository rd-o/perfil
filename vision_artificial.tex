\section{Antecedentes}
La visión artificial es un subcampo de la inteligencia artificial. El propósito de la visión artificial es que un sistema programado con esta tecnología pueda \guillemotleft entender\guillemotright  las imágenes que capta de su entorno. Un robot autónomo tiene un comportamiento y realiza tareas con alto grado de autonomía, esto es particularmente necesario en exploración espacial, fabricación, limpieza, etc.

Un robot o entidad virtual, para poder desenvolverse en su entorno, necesita información de entrada, que debe ser procesada para luego activar sus actuadores de acuerdo a los datos de entrada procesados. Estos datos de entrada pueden ser datos de distintos tipos de sensores. Dependiendo la complejidad del robot, estos pueden ser de sonido, imágenes y tacto. En cuanto a los sensores de imágenes y vídeo, para poder ser procesados es necesario tener un sistema que maneje estas imágenes y luego entienda el contenido de cada una de las imágenes que se le presente.

El proyecto estará orientado a desarrollar un sistema de navegación autónomo usando las bibliotecas de OpenCV (Open Source Computer Vision) para un robot. El robot a construir es un tipo, con dos orugas. Este tipo de configuracion permite al robot una movilidad en todas las direcciones, y tambien en terrenos de tierra, grava y arena.

El robot necesita una plataforma independiente para poder desenvolverse, por lo tanto se usara un ordenador de placa unica de bajo costo y bajo consumo de energia. 

El proyecto estara orientado al area de inteligencia artificial y robotica.

\section{Identificación del problema}
%efectos: 
% - el robot no es totalmente autonomo
% - el robot tiene dificultad para desplazarse en lugares muy alejados
% - el robot tiene dificultad para desplazarse en lugares de dificil acceso
% - 

%causas:
% - un operador debe operar al robot para que pueda desplazarse en su entorno
% - La distancia entre el operador y el robot es muy grande
% - No existe vision directa entre el operador y el robot 

La falta de un sistema que permita a un robot poder desplazarse de forma autónoma, en un entorno alejado o de difícil acceso.

\subsection{Definición del problema}

Un \emph{entorno alejado} puede estar a cientos o miles de kilómetros alejado de los operadores. En entornos extremadamente alejados normalmente es difícil poder controlar un robot en tiempo real, ya que las señales enviadas pueden tardar en llegar a su destino.

Un \emph{entorno de difícil acceso} puede ser un pasadizo o cueva de difícil acceso para un operador. En este tipo de entorno puede que no haya visión directa desde el operador hacia el robot, por este motivo es casi imposible poder operar el robot. En ciertos casos en que se tenga un robot con una cámara inalámbrica, puede que la señal de control sea muy baja, debido a obstáculos y distancia alejada, esto puede ocasionar la perdida de la señal de vídeo y control.

\section{Objetivo del proyecto}
\subsection{Objetivos de la aplicación}
\begin{itemize}
  \item Se diseñara un robot que sea capaz de soportar la plataforma de hardware necesaria para el sistema.
  \item Se diseñara un robot de tal manera que sea totalmente inalambrico.
  \item Se implementara un sistema de vision artificial que sea capaz de reconocer obstaculos al frente del robot.
  \item Se implementara un sistema de navegacion que permita al robot moverse de forma autonoma a un destino establecido.
  \item Se implementara un sistema de vision artificial que sea capaz de distinguir obstaculos existentes en un plano.
  \item Se implementara un sistema de proteccion que alarme al robot en caso de estar peligrosamente inclinado
  \item Se implementara un sistema de comunicaciones, para que el robot pueda recibir las coordenadas de destino y otros datos. 
  \item Se implementara un algoritmo para detectar la profundidad de los obstaculos.

\end{itemize}

\subsection{Objetivos metodológicos}
Los objetivos a cumplir durante el desarrollo del proyecto son:
\begin{itemize}
  \item Se estimaran los costos y tiempo de desarrollo del proyecto.
  \item Se realizara investigacion acerca de las tecnologias relacionadas al proyecto.
  \item Se realizara una estimacion de cada historia de usuario del product backlog.
  \item Se asignara una prioridad a cada historia de usuario, para luego ejecutar primero las historias con mas prioridad.
  \item 
\end{itemize}

\section{Alcance}
El proyecto tendra el siguiente alcance:
\begin{itemize}
  \item El robot podra desplazarse en un terreno con obstaculos, y sera capaz de esquivar obstaculos o pasarlos por encima.
  \item El robot podra desplazarse por terrenos planos, y tambien por pendientes no mayores a 25 grados. 
  \item El software de navegacion recibira coordenadas para llegar a un destino establecido por un operador, y este software sera capaz de conducir al robot a su destino de forma autonoma.
  \item El robot contara con una camara de video, la imagen obtenida por esta sera procesada por el sistema de vision artificial.
  \item Por limitaciones del hardware a utilizar, las obstaculos del terreno tienes que tener colores diferentes al terreno, para falilitar el trabajo al sistema de vision artificial.
  \item La cobertura entre el operador y el robot estara limitada por el maximo alcance del sistema de comunicacion inalambrica a utilizar.
  \item El sistema de vision artificial operara solo en ambientes con iluminacion suficiente como para poder distinguir los obstaculos.
\end{itemize}

\section{Justificación}
Existen diferentes aplicaciones en las que este proyecto puede ser util, por ejemplo en exploracion espacial. En estos casos el robot esta alejado de su operador por distancias astronomicas, por tanto, el control del robot no es en tiempo real, y ademas hay posibilidad de que las comunicaciones se corten de vez en cuando, dejando al robot estatico o incluso en riesgo de dejarlo inutilizable.

Otra aplicacion para este proyecto es la exploracion de lugares de dificil acceso para una persona, estos lugares pueden ser cuevas o lugares con contaminacion de materiales radioactivos.

En nuestro pais dificilmente se pueden encontrar proyectos relacionados a la inteligencia artificial y robotica. Este proyecto de grado pretende aportar a la comunidad universitaria con investigacion relacionada a las areas de este proyecto.
